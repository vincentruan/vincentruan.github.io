# vincentruan.github.io

Vincent Ruan 的个人技术博客，记录在软件工程、后端架构、前端开发等领域的学习与思考。

## 项目简介

这是一个个人技术写作与知识沉淀的站点。博客文章涵盖编程语言、基础设施、工具链、工程实践等主题，旨在分享技术心得、整理学习笔记，并作为个人成长的长期存档。

站点已部署上线，可通过 GitHub Pages 访问。

## 本地预览

```bash
npm install
npm run server      # 启动本地预览服务
npm run build       # 生成静态站点
npm run clean       # 清理构建产物
```

> 需要 Node.js 20.19.0 或更高版本。

## 目录结构

- `source/_posts/` — 博客文章（Markdown）
- `source/` — 静态资源与自定义页面
- `_config.yml` — 站点配置
- `_config.next.yml` — 主题配置
- `scaffolds/` — 文章模板
- `scripts/` — 构建期自定义脚本
